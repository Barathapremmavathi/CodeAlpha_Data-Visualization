# 📊 Data Visualization Internship Project

## 🚀 Overview
This project transforms raw Superstore sales data into meaningful business insights using Python. It focuses on data cleaning, visualization, and storytelling to support better decision-making.

---

## 🎯 Objective
- Clean and preprocess real-world dataset  
- Analyze sales and profit performance  
- Create meaningful visualizations  
- Extract business insights from data  

---

## 🛠️ Tools Used
- Python 🐍  
- Pandas 📊  
- Matplotlib 📈  
- Seaborn 🎨  
- Jupyter Notebook 📓  

---

## 📂 Dataset
Superstore Sales Dataset (CSV/Excel format)  
Includes:
- Sales  
- Profit  
- Category  
- Region  
- Product details  

---

## 📊 Visualizations Included
- 📈 Sales Trend Over Time  
- 📊 Sales by Category  
- 💰 Profit by Category  
- 🥧 Profit Distribution  
- 🛒 Top 10 Products  
- 🌍 Region-wise Sales  

---

## 🧠 Key Insights
- Technology category generates highest profit  
- Sales show seasonal growth patterns  
- Few products contribute most revenue  
- Regional sales performance varies significantly
